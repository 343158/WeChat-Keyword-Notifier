# 微信关键词提醒器

一个用于监听微信通知并在检测到指定关键词时提醒用户的安卓应用。

## ✨ 功能特点

- ✅ **无需 Root**：使用 Android 官方通知监听 API
- ✅ **关键词匹配**：监控微信通知中的指定关键词
- ✅ **实时提醒**：匹配到关键词时发送高优先级通知
- ✅ **动态配置**：可在应用内添加/删除关键词
- ✅ **默认关键词**：红黄土、干杂土、代运、放飞、红砖渣

## 🚀 使用方法

### 方式一：GitHub Actions 在线编译（推荐）

1. **上传代码到 GitHub**
   - 在 GitHub 创建新仓库
   - 上传本项目所有文件

2. **自动编译**
   - 推送代码后，GitHub Actions 会自动编译
   - 在 Actions 页面查看编译进度

3. **下载 APK**
   - 编译完成后，在 Actions 页面下载 APK 文件
   - 安装到安卓手机

### 方式二：本地编译

1. **安装 Android Studio**
   - 下载地址：https://developer.android.com/studio
   - 安装完成后打开项目

2. **编译 APK**
   - 点击菜单：`Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
   - 编译完成后在 `app/build/outputs/apk/debug/` 目录找到 APK

3. **安装到手机**
   - 将 APK 文件传输到安卓手机
   - 在手机上安装（需要允许安装未知来源应用）

## 📱 使用步骤

### 1. 授权通知监听权限

首次打开应用会提示授权通知监听权限：

1. 点击"确定"
2. 在系统设置中找到"微信关键词提醒器"
3. 开启权限

### 2. 配置关键词

- **添加关键词**：在输入框输入关键词，点击"添加关键词"
- **删除关键词**：在输入框输入要删除的关键词，点击"删除关键词"
- **查看列表**：当前所有关键词会显示在列表中

### 3. 开始监控

授权并配置完成后，应用会在后台自动监听微信通知：

- 当微信收到新消息时，应用会检查内容
- 如果包含关键词，会发送高优先级通知提醒你
- 通知会显示：群名/联系人、消息内容、匹配的关键词

## ⚙️ 默认关键词

应用首次启动时会自动添加以下默认关键词：

- 红黄土
- 干杂土
- 代运
- 放飞
- 红砖渣

你可以根据需要在应用内添加或删除关键词。

## 🔧 故障排除

### 问题：收不到提醒

**解决方法**：

1. **检查通知监听权限**
   - 设置 → 应用管理 → 微信关键词提醒器 → 权限 → 通知监听 → 开启

2. **检查微信通知设置**
   - 微信 → 我 → 设置 → 新消息通知 → 确保"通知显示消息详情"已开启

3. **检查关键词配置**
   - 打开应用，确认关键词列表中有你想要监控的关键词

### 问题：应用被系统杀死

**解决方法**：

1. **添加到白名单**
   - 设置 → 电池管理 → 微信关键词提醒器 → 允许后台运行

2. **关闭省电模式**
   - 部分手机会自动清理后台应用，需要手动添加到白名单

## 📋 技术说明

- **最小 SDK**：Android 7.0 (API 24)
- **目标 SDK**：Android 14 (API 34)
- **权限**：通知监听服务（无需 Root）
- **核心技术**：NotificationListenerService
